# Mental Health Chatbot Datasets

Add text files here to expand your chatbot's knowledge base.

## How to Add Datasets

1. Create a `.txt` file with mental health information (e.g., `depression.txt`, `anxiety.txt`, `sleep.txt`)
2. Add clear sections with relevant keywords
3. Place the file in this `data/` folder
4. The chatbot will automatically load and use all text files

## Dataset Format

```
TOPIC TITLE

1. Main Concept
Description with key information.

2. Common Signs
• Bullet point 1
• Bullet point 2

3. How to Help
Clear, actionable steps.
```
